# MotorCore-V-0.1-
MotorC es un proyecto orientado a la gestión de reparación, repuestos y control operativo para motocicletas, pensado para talleres mecánicos, centros de servicio y administradores técnicos que necesitan una solución visual, moderna y organizada.
El sistema simula un dashboard administrativo con áreas funcionales, gestión visual de información y una interfaz inspirada en conceptos de UI/UX modernos y técnicos.

Objetivo del Proyecto:

Centralizar la gestión de reparaciones de motocicletas
Facilitar el control de repuestos
Simular un entorno administrativo (admin + usuario logueado)
Presentar la información en un dashboard visual y responsivo
Servir como base escalable para futuras integraciones (backend, base de datos, APIs)


Funcionalidades Principales

Dashboard principal con tarjetas informativas
Sidebar con áreas del sistema (enlaces futuros)
Encabezado con:

Logo del proyecto
Nombre del sistema
Usuario activo (simulado)
Barra de búsqueda funcional (frontend)


*Diseño responsivo
Fondo visual translúcido integrado al dashboard
Estilo orientado a gestión técnica y mecánica
